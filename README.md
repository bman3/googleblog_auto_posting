# Auto Blog Posting System

자동 블로그 포스팅 시스템입니다.

## 기능

- Google Trends 키워드 자동 생성
- Gemini AI로 블로그 포스트 생성
- Hugging Face로 AI 이미지 생성
- Blogger API로 자동 포스팅
- GitHub Actions로 자동 배포

## 테스트 배포

이 파일이 수정되면 GitHub Actions가 자동으로 배포를 시작합니다!
